Les classes sont réservées a un usage répété sur la page ex:

<img class="icon" />
quand tu as plusieurs Icones

Dans un cas unique, on preferera les ID

<img id="MonImageUnique" />


Pour les images utiliser 

object-fit: contain | cover

cover: estéthique 
contain: Valeur d'affichage importante ex logo dans un coin de l'image

laisser la hauteur gérée auto par la taille de l'image ou contraindre mais en utilisant object-fit
